# Source Map

This skill adapts guidance from Toss's `technical-writing` repository:

https://github.com/toss/technical-writing

The source repository is licensed under CC BY-NC-SA 4.0. This project keeps Toss-derived writing guidance under the same content license and provides attribution in `NOTICE.md`.

## Adaptation Map

| This repository | Toss source area | Adapted guidance |
| --- | --- | --- |
| `references/document-types.md` | `docs/type/*`, `docs/tutorial/*` | Learning, problem-solving, reference, explanation, and article-like document selection |
| `references/structure-principles.md` | `docs/architecture/*` | One topic per page, value first, headings, overview, predictability, and background |
| `references/sentence-style.md` | `docs/sentence/*` | Clear subject, compactness, concreteness, natural Korean expression, and terminology consistency |
| `references/review-rubric.md` | `docs/tutorial/review-prompt.mdx` plus checklist pages | Checklist-based review flow and severity labels |
| `references/prompt-workflows.md` | `docs/tutorial/review-prompt.mdx` | AI-assisted drafting, structuring, and review prompts |
| `assets/templates/*` | `docs/type/index.md`, `docs/tutorial/structure.mdx` | Reusable document skeletons |

## Adaptation Rules

- Do not copy Toss pages as full verbatim mirrors.
- Translate and condense guidance into agent instructions.
- Preserve attribution and the CC BY-NC-SA 4.0 license for content.
- Keep scripts and tests separate under MIT.
